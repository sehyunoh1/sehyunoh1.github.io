To control the sppeed of Testimonial carousel just open up main.js file and search for  // Testimonial Carousel and change the following control

Code to edit:

    $("#testimonial-carousel").owlCarousel({
        // Testimonial Carousel Settings
        navigation: true, // Show next and prev buttons
        navigationText: ["<i class='fa fa-chevron-left'></i>", "<i class='fa fa-chevron-right'></i>"],
        slideSpeed: 1000, //Change Slide Speed
        paginationSpeed: 1000, // Change Pagination Speed
        singleItem: true,
        pagination: false,
        autoPlay: true,

Google Map

To Intrigate your locataion using Google Map and Show your location, Open Up the main.js file and search for // Google Map toggle and edit the follwing code with your own Latitude and Longitude . To genatrate your lat and long follow this Tutorial

Code to Edit:

                        map = new GMaps({
                        el: '#gmap-wrapper',
                        lat: 22.3590557, //Replace Here Location Google Map Latitude
                        lng: 91.8213111, //Replace Here Location Google Map Longitude
                        scrollwheel: false,
                        zoom: 17,
                        zoomControl: true,
                        panControl: true,
                        streetViewControl: true,
                        mapTypeControl: false,
                        overviewMapControl: false,
                        clickable: true
                    });

Change Template Color

To change the template color just open up your desiered template and search for <!-- Color Styles --> and delete all the color stylesheet witout the color you want to keep.

Code To Edit:

        <!-- Color Styles | To Chage the color simply remove all the stylesheet form bellow without the color you want to keep -->
